Dynamic Mode Toolbox - Tutorial¶

Installation¶

To install dmd, simply run the following command (we advise you to do so in your favorite virtual environment)

pip install dmd

Importing the adequate modules¶

For this tutorial, we will load both the Decomposition module as well as the different plotting classes.

[17]:

from dmd import Decomposition, Radar, TimePlot, Brain, Spectre

The next step consists in loading adequate data for the creation of a Decomposition instance.

[2]:

dcp = Decomposition(filenames='/home/adhaene/Documents/Dynamic Mode Toolbox Data/Rest1LR_Sub100307_Glasser.mat')

[18]:

Radar(dcp.df, dcp.atlas).figure().show()

[7]:

Brain(dcp.df, 4, dcp.atlas.coords_2d).figure(colormap='RdBu')

[15]:

Spectre(dcp.df, ['Group']).figure()

[16]:

TimePlot(dcp.df).figure()
